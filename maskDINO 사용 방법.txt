2. 등록
- maskDINO 자체 내장된 등록 파일은 coco 데이터셋에 지나치게 특화되어 사용이 어려움.
- 새로 register 파일을 만들고, train_net.py 에서 등록 옵션 추가 / 내장된 등록파일 수정
- detectron2에 내재된 detectron2/data/coco_panoptic.py 의 register_coco_panoptic 함수 이용


3. config 파일
- config 파일에서 클래스 수, 학습률, 학습 횟수 등 조정
- semantic, instance 평가도 진행할 것인지 옵션 조정 가능 
MaskDINO/configs/coco/panoptic-segmentation/maskdino_R50_bs16_50ep_3s_dowsample1_2048.yaml 


4. 사전학습 가중치
- 깃허브에서 사전학습 가중치 다운로드
- 유의점: Backbone 선택 필수 ( R50 : 성능 약간 낮음, 파인튜닝 빠름 Swin-L : 성능 우수, 파인튜닝 느림)


[8. 학습]
python train_net.py --num-gpus 1 --config-file configs/coco/panoptic-segmentation/maskdino_R50_bs16_50ep_3s_dowsample1_2048.yaml MODEL.WEIGHTS pretrained/maskdino_swinl_50ep_300q_hid2048_3sd1_panoptic_58.3pq.pth


[9. 평가모드]
python train_net.py --eval-only --num-gpus 1 --config-file configs/coco/panoptic-segmentation/maskdino_R50_bs16_50ep_3s_dowsample1_2048.yaml MODEL.WEIGHTS output/model_final.pth

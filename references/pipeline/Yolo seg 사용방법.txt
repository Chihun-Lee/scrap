0. 가상환경 생성

1. yolo 다운 후 패키지 설치
파이썬 3.10.12 버전

pip install -r requirements.txt
pip install ultralytics
pip install tqdm
pip install pycocotools
pip install pandas
pip install numpy==1.26.4
pip install "git+https://github.com/cocodataset/panopticapi.git"

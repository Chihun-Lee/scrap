## 한 줄 결론
수정방안(대표클래스 병합·additive-only·Point/크기 컷오프)은 ver1 실패원인을 정확히 겨냥해 **공학적으로 타당**. 핵심 GAP은 **클래스 불균형(연구항목 누락)**, 보완점은 **불확실-case 처리·dense pile ignore영역·modal 명시·PQ 겹침 해소**. 상세는 분석 문서 참조.
